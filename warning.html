ODT - Omni Development Terminal
Offline AI Development Dashboard with Local LLM
📋 Quick Setup Checklist
1
Install Node.js
Required for running the proxy server. Download from nodejs.org
node --version
2
Install Ollama
Required for running local AI models. Download from ollama.ai
ollama --version
3
Install Dependencies
Run npm install in this folder
npm install
4
Download a Model
Pull a model for offline use (choose one):
ollama pull phi:2.7b # Fast ollama pull qwen2.5:7b # Recommended
5
Start ODT
Start Ollama, then the proxy server, then open this file
ollama serve npm start
