MODELS NOT SHOWING?

1. Check proxy is running:
   - Look for "localhost:9001" in console
   - Verify: Green proxy indicator

2. Verify models exist:
   - Open File Explorer
   - Navigate to: E:\models\
   - Should see: Qwen_*.gguf file

3. Try manual scan:
   - Click "🔍 Scan E:\models" button
   - Wait for results
   - Check browser console (F12) for errors

4. Restart proxy:
   - Close: Proxy console window
   - Run: node proxy.js again
   - Refresh: Dashboard (F5)

PROXY NOT CONNECTING?

1. Check if already running:
   - Try: netstat -ano | findstr "9001"
   - Kill: taskkill /IM node.exe /F
   - Restart: node proxy.js

2. Port might be in use:
   - Check: What's on port 9001?
   - Free port: run node proxy.js on different port
   - Update: index.html PROXY_HOST constant

AUTO-REFRESH NOT WORKING?

1. Check browser console:
   - F12 to open
   - Look for errors
   - Check Network tab for /api/scan-models call

2. Verify location.reload() added:
   - Check: index.html openModelsModal() function
   - Should show: location.reload() call

3. Try manual refresh:
   - F5 to refresh page
   - Click Models button again
   - Check if models appear
