# TEoAAAG Dashboard App

## Quick Start

1. Double-click launch.bat
2. Wait for proxy to start
3. Dashboard opens in browser

## Files

- index.html - Main dashboard UI
- proxy.js - Node.js CORS proxy for Ollama
- launch.bat - Launcher script
- CONFIG.env - Configuration
- package.json - Node.js metadata

## Requirements

- Node.js (for proxy)
- Ollama running on localhost:11434
- No Docker required

## Paths

All paths are E: drive only. Fully standalone.

## Config

Edit CONFIG.env to change:
- OLLAMA_HOST
- PROXY_PORT
- Model selection
