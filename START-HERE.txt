================================================================================

STEP 1: SETUP (First time only)
  Double-click: setup-lib-system.bat
  This initializes the lib/ directory and bundle

STEP 2: OPEN LIBRARY MANAGER
  Double-click: lib-manager.bat
  Or: lib-manager.bat ui (for Windows GUI)

STEP 3: AUTO-BUNDLE
  Choose option: 1 (Auto-Bundle Scripts)
  Or in GUI: Click "📦 Bundle All"
  Wait for complete message

STEP 4: TEST DASHBOARD
  Open browser: file:///E:/dashboard-app/index.html
  Press F12 for console
  Should show: "[TEoAAAG] Library bundle loaded: 1 libraries"

STEP 5: CREATE YOUR FIRST LIBRARY
  Create file: my-feature.lib.js
  Add this code:
    function hello() {
        console.log('Hello from my library!');
    }
  Run bundler (Step 2-3)
  Test in console: hello()

DONE! You now have a working library system.

================================================================================
                         HOW IT WORKS (Simple)
================================================================================

BEFORE (Old Way):
  index.html includes multiple .js files:
    <script src="utility.js"></script>
    <script src="gordon-testbench.js"></script>
    <script src="analytics.js"></script>
    (lots of files, hard to manage)

AFTER (New Way):
  index.html includes ONE file:
    <script src="./lib/bundle.lib.js"></script>
    
  lib/bundle.lib.js contains ALL your code:
    ✓ Combines all .lib.js files
    ✓ Deduplicates functions
    ✓ Validates dependencies
    ✓ Single HTTP request
    ✓ Better performance

HOW YOU USE IT:
  1. Create: my-feature.lib.js (in E:\dashboard-app\)
  2. Run: lib-manager.bat auto
  3. All functions auto-available in dashboard
  4. No manual script tags needed

                        CREATING YOUR LIBRARY
================================================================================

TEMPLATE:

  // Feature: My Feature Name
  // Name: my-feature.lib.js
  // Description: What this does

      console.log('Hello from my-feature library!');
  }

  console.log('[TEoAAAG] My Feature library loaded');

INSTRUCTIONS:

  1. Create file: my-feature.lib.js
     In folder: E:\dashboard-app\

  2. Write your code (use template above)

  3. Save file

  4. Run bundler:

     Choose: 1 (Auto-Bundle)

  5. Test in dashboard:
     F12 console
     Type: myFunction()
     Should work!

NAMING RULES:

================================================================================
                         TROUBLESHOOTING
================================================================================

"I don't see my library functions"
  → Run: lib-manager.bat auto
  → Reload dashboard: Ctrl+F5 (hard refresh)
  → Check console: F12

"Bundle didn't create"
  → Check: lib/ folder exists
  → Check: .lib.js files exist
  → Run: setup-lib-system.bat
  → Try again: lib-manager.bat auto

"Duplicate function error"
  → Check: Two .lib.js files have same function name
  → Solution: Rename one function
  → Run: lib-manager.bat auto again

"Function not found"
  → Check: Spelling matches exactly
  → Check: Library is .lib.js format
  → Check: File is in E:\dashboard-app\
  → Run: lib-manager.bat check

"GUI won't open"
  → Try: Run as Administrator
  → Try: powershell -File lib-manager-ui.ps1
  → Check: PowerShell execution policy
